# SIEM com Python e IA

Projeto baseado no artigo de Chaitanya Rahalkar.

Nesse projeto foi realizado um script de SIEM (Security Information and Event Management) ou Gerenciamento
de Eventos e Segurança da informação, sendo uma solução que coleta, agrega e analisa dados de várias fontes para detectar, investigar e responder a ameaças de segurança em um ambiente de TI.

Fonte: [https://www.freecodecamp.org/news/how-to-create-a-python-siem-system-using-ai-and-llms/](https://www.freecodecamp.org/news/how-to-create-a-python-siem-system-using-ai-and-llms/)

# Criptografia

Este repositório contém laboratórios e exemplos relacionados a criptografia.

## Diferença entre SHA256, AES e RSA

- **SHA256**: É uma função hash criptográfica, utilizada para gerar um resumo fixo (256 bits) de uma mensagem. Não é usada para criptografar ou descriptografar dados, mas sim para garantir integridade, pois pequenas alterações na entrada geram saídas completamente diferentes.

- **AES**: É um algoritmo de criptografia simétrica, ou seja, utiliza a mesma chave para criptografar e descriptografar dados. É amplamente utilizado para proteger grandes volumes de dados devido à sua eficiência e segurança.

- **RSA**: É um algoritmo de criptografia assimétrica, utilizando um par de chaves (pública e privada). É comumente usado para troca segura de chaves e assinatura digital, mas não é eficiente para criptografar grandes quantidades de dados.

Cada algoritmo tem um propósito específico dentro da criptografia: SHA256 para integridade, AES para confidencialidade de dados em massa e RSA para troca segura de informações e autenticação.
